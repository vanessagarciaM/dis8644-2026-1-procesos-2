## Como se realizan nuestros sueños?

1. Para exportar nuestra placa PCB, se debe hacer en formato gerber.

2. Para esto necesitamos como mínimo exportar 7 capas, en este caso:
1. 2 de cobre
2. 2 de silkscreen
3. 2 de máscara
4. contorno

3. Se hace con el botón de plot arriba a la izquierda
4. En la ventana que aparezca: A la izquierda elijo que capas seleccionaré
5. Se guarda en una carpeta
6. Hay que hacer 2 clicks: Plot y generar archivos de taladrado
7. Se generan muchos archivos
8. En visor gerner se puede ver si es coherente o no lo que hay en el archivo

> Más cuidado con curbir los nombres de los componentes con la silkscreen

9. Comprimir el archivo y listo
10. Luego fueron a JLCPCB y subimos nuestro archivo gerber (.gbr)
11. En la ventana de cotizar no solamente nos va a decir cuanto vale la PCB, Este es el primer checkeo para confirmar que n uestroa rchivo está bien.
12. Podemos presionar gerber view para ver las capas y la placa en  3D
13. Primero bacan, tenemos las medidas correctas.
14. Se pide la cantidad que queremos
15. Se escoge el color
16. acabado de la placa HASL es el estándar hay con y sin plomo (sin plomo salud es importante)
