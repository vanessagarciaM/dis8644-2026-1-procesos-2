# Resumen de Controles Básicos — KiCad

## Controles Generales
| Tecla | Función | Descripción |
|---|---|---|
| `Ctrl + S` | Guardar | Guarda el proyecto o los cambios realizados. |
| `Esc` | Cancelar / Volver al mouse | Sale de una herramienta o acción activa y vuelve al cursor normal. |

---

## ¿Como usar los Componentes?
| Tecla | Función | Descripción |
|---|---|---|
| `M` | Mover | Permite mover un componente seleccionado libremente. |
| `G` | Arrastrar conjunto | Mueve el componente junto con sus conexiones y pistas asociadas. |
| `R` | Rotar | Rota el componente seleccionado. |
| `X` | Reflejar en eje X | Invierte o espeja el componente respecto al eje X. |
| `V` | Cambiar valor | Modifica el valor o nombre del componente seleccionado. |

---

## Conexiones
| Elemento | Función | Descripción |
|---|---|---|
| `Puntos` | Unión eléctrica | Indican que dos líneas o cables están conectados eléctricamente. |

---

## Menú de Componentes
El menú de componentes permite:
- Agregar componentes electrónicos.
- Buscar símbolos.
- Editar propiedades.
- Seleccionar librerías.
- Insertar elementos al esquema electrónico.
